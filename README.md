### ✨ Tentang Proyek
**Campus 'D Mart** adalah platform manajemen lapak wirausaha mahasiswa yang dirancang untuk memudahkan transaksi, monitoring stok, serta pengelolaan keuangan koperasi kampus secara transparan dan efisien.

---

### 🚀 Fitur Unggulan
*   **Multi-Role Dashboard**: Akses terpisah untuk Pembeli, Seller, dan Admin Koperasi[cite: 4, 5].
*   **Kampus Ad-Center**: Fitur promosi eksklusif bagi Seller PRO untuk meningkatkan visibilitas produk[cite: 1, 2].
*   **Real-time Monitoring**: Pemantauan stok tenant dan akumulasi omzet koperasi secara *live*[cite: 2, 5].
*   **Transparansi Keuangan**: Laporan LPJ, CaLK, dan Neraca yang dapat dicetak secara formal[cite: 1, 2, 5].
*   **Keamanan Berlapis**: Proteksi akses berbasis peran dan email untuk menjaga integritas data[cite: 1, 2].

---

### 🛠 Tech Stack
- **Framework**: Laravel
- **Styling**: Tailwind CSS
- **Database**: MySQL
- **Environment**: Windows Local Development
